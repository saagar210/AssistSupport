# GitHub Topics for AssistSupport

To add topics to the repository:

1. Go to: https://github.com/saagar210/AssistSupport/settings/general
2. Scroll to "Repository topics"
3. Add these topics:
   - `ai-support` - AI-powered support
   - `offline-first` - All processing local
   - `hipaa-compliant` - Healthcare compliance
   - `gdpr-compliant` - Privacy regulation
   - `it-support` - IT support focus
   - `jira-integration` - Jira connectivity
   - `llm` - Large language model
   - `tauri` - Framework used
   - `rust` - Backend language
   - `react` - Frontend framework

These topics help IT engineers discover AssistSupport when searching for:
- "offline AI"
- "HIPAA compliant support"
- "IT ticket response AI"
- "Local LLM support"
