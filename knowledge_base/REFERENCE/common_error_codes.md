# Common Error Codes and Resolutions

## VPN Errors

| Code | Message | Resolution |
|------|---------|------------|
| GP-001 | "Portal unreachable" | Check internet connection, try different network |
| GP-002 | "Authentication failed" | Verify credentials, check MFA, reset password if needed |
| GP-003 | "Certificate expired" | Contact IT to renew device certificate |
| GP-004 | "Gateway unavailable" | Try alternate gateway, check VPN status page |

## SSO / Authentication Errors

| Code | Message | Resolution |
|------|---------|------------|
| SSO-401 | "Unauthorized" | Clear browser cache, try incognito, re-authenticate |
| SSO-403 | "Access denied" | Request access through Access Portal |
| SSO-429 | "Too many attempts" | Wait 30 minutes or contact IT for unlock |
| MFA-001 | "MFA challenge failed" | Sync authenticator app clock, try SMS code |

## Email Errors

| Code | Message | Resolution |
|------|---------|------------|
| SMTP-550 | "Recipient rejected" | Verify email address, check for typos |
| SMTP-552 | "Message too large" | Reduce attachment size, use ShareFile for large files |
| ENC-001 | "Encryption failed" | Ensure recipient has valid certificate |

## General

| Code | Message | Resolution |
|------|---------|------------|
| DISK-001 | "Disk space low" | Clear temp files, move files to OneDrive, contact IT |
| NET-001 | "No network" | Check Wi-Fi, restart adapter, try wired connection |
| PRINT-001 | "Printer offline" | Check printer power, restart print spooler service |
