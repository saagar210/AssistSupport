# Supported Software List

## Productivity
- Microsoft 365 (Word, Excel, PowerPoint, Outlook, Teams)
- Adobe Acrobat Pro
- Slack (approved for cross-team communication)
- Zoom (approved for external meetings)

## Development
- Visual Studio Code
- JetBrains IDEs (IntelliJ, PyCharm, WebStorm)
- Docker Desktop
- Git (command line + GitHub Desktop)
- Node.js (LTS versions)
- Python 3.10+
- Postman

## Security
- CrowdStrike Falcon (mandatory endpoint protection)
- GlobalProtect VPN (mandatory for remote access)
- 1Password Business (approved password manager)

## Design
- Figma
- Adobe Creative Cloud (by request)

## Not Supported / Prohibited
- Personal antivirus software (conflicts with CrowdStrike)
- Unauthorized VPN clients
- Torrent clients
- Cryptocurrency miners
