# Password and Authentication Policy

**Status:** ACTIVE
**Classification:** MANDATORY
**Last Updated:** 2025-01-05

## Policy Statement

All company accounts must use strong passwords and multi-factor authentication (MFA). Password sharing is strictly forbidden.

## Password Requirements

- Minimum 12 characters
- Must contain uppercase, lowercase, number, and special character
- Cannot reuse last 12 passwords
- Maximum age: 90 days
- Cannot contain username, name, or common dictionary words

## Multi-Factor Authentication

MFA is **REQUIRED** for:
- All SSO logins
- VPN access
- Admin console access
- Cloud service access
- Remote desktop connections

## Prohibited

- Sharing passwords with anyone (including IT support)
- Storing passwords in plain text files
- Using the same password across work and personal accounts
- Writing passwords on physical media (sticky notes, whiteboards)
