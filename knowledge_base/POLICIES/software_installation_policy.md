# Software Installation Policy

**Status:** ACTIVE
**Classification:** MANDATORY
**Last Updated:** 2025-01-10

## Policy Statement

Only IT-approved software may be installed on company devices. Users do NOT have local admin rights and must submit a software request through the IT portal.

## Approved Software

Software must be on the company-approved list. Check the IT portal for the current catalog.

## Request Process

1. Submit request via IT Self-Service Portal
2. Manager approval required
3. IT reviews for security/licensing compliance
4. If approved, IT deploys remotely via SCCM/Intune
5. Typical turnaround: 2-3 business days

## Prohibited

- Installing software without IT approval
- Using portable/standalone executables to bypass restrictions
- Installing browser extensions not on the approved list
- Running scripts or tools from the internet without approval
