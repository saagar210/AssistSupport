# Approved Cloud Storage Policy

**Status:** ACTIVE
**Classification:** MANDATORY
**Last Updated:** 2025-01-12

## Policy Statement

Only company-approved cloud storage services may be used for work data. These services provide encryption, audit logging, and access controls required by compliance.

## Approved Services

| Service | Use Case | Max File Size |
|---------|----------|---------------|
| OneDrive for Business | Primary file storage and sharing | 250GB |
| SharePoint Online | Team/department collaboration | 250GB |
| Google Drive (Workspace) | Cross-team collaboration | 5TB |
| Dropbox Business | External partner sharing | 50GB |
| ShareFile | Large/sensitive file transfers | 100GB |

## Features of Approved Services

- End-to-end encryption in transit and at rest
- Full audit trail of all access and modifications
- Granular access controls and expiring links
- Data Loss Prevention (DLP) scanning
- HIPAA/GDPR/SOX compliant
- IT-managed retention policies

## Prohibited

- Personal cloud storage accounts (personal Dropbox, iCloud, Google Drive)
- Unauthorized file sharing services (WeTransfer, MediaFire, etc.)
- Peer-to-peer file sharing
- Removable media as an alternative to cloud storage
