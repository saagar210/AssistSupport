# Remote Access Policy

**Status:** ACTIVE
**Classification:** MANDATORY
**Last Updated:** 2025-01-08

## Policy Statement

Remote access to company resources is permitted only through company-approved methods with proper authentication and encryption.

## Approved Remote Access Methods

- Company VPN (GlobalProtect) — required for all remote work
- Azure Virtual Desktop (AVD) — for accessing internal applications
- Microsoft 365 web apps — accessible via SSO with MFA

## Prohibited

- Personal VPN services
- Third-party remote desktop tools (TeamViewer, AnyDesk) unless IT-approved
- Direct SSH to internal servers from outside VPN
- Port forwarding or tunneling without IT approval
- Accessing company resources from public Wi-Fi without VPN
